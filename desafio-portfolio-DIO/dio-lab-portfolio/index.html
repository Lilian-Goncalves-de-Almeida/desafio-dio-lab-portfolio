Cursos

Digital Innovation One. Jul 2024. 87h.

Curso de 87 horas focado no desenvolvimento de soluções Back-End utilizando Java e Spring Boot, com ênfase em práticas de banco de dados local e controle de versão com Git e GitHub. Os alunos aprenderam a criar APIs robustas e escaláveis, integrar bancos de dados para armazenamento e manipulação de dados, além de aplicar práticas de versionamento e colaboração no desenvolvimento de software. Ideal para quem deseja se aprofundar no desenvolvimento de sistemas e soluções back-end.

Digital Innovation One. Jan 2024. 17h.

Curso de 77 horas focado no ensino de HTML, CSS e JavaScript para criação de jogos 2D interativos. Através de uma abordagem prática, os alunos aprendem a estruturar páginas, estilizar elementos e programar interatividade, com ênfase em projetos temáticos do universo gamer. Ideal para iniciantes, o curso oferece uma base sólida em desenvolvimento web, capacitando para a criação de jogos e proporcionando habilidades para ingressar na área de tecnologia e desenvolvimento de jogos.

Ada Tecnologia e Educação. Mar 2023. 17h.

Curso de 17 horas abordando as principais ferramentas e conceitos de Data Science. Os alunos aprenderam a utilizar Git e GitHub para versionamento de código, trabalhar com bancos de dados para armazenamento e consulta de dados, aplicar técnicas de estatísticas para análise e interpretação de dados, e explorar o uso do Power BI para visualização e criação de relatórios interativos. Ideal para iniciantes que buscam desenvolver habilidades essenciais em ciência de dados.

Secretaria de Ciência e Tecnologia de João Pessoa. Fev 2021. 30h.

Curso de 30 horas focado no aprendizado de BPMN (Business Process Model and Notation) para a modelagem e gestão de processos empresariais. Os alunos adquiriram conhecimentos sobre a estrutura e os símbolos do BPMN, além de aprender a criar diagramas eficientes para mapear, analisar e otimizar processos organizacionais. O curso é ideal para quem deseja aprimorar a gestão de processos e implementar melhorias no fluxo de trabalho dentro de uma empresa.
